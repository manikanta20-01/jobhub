## Job Description

CodeCraft Technologies is a product development company building intelligent software solutions. We're hiring Python Developers to work on backend systems, APIs, and AI integrations.

You'll develop RESTful APIs, work with databases, integrate machine learning models, and build scalable microservices. Our projects span fintech, healthcare, and e-commerce domains.

The team follows agile methodologies with emphasis on code quality, testing, and continuous learning. Regular tech talks and hackathons keep the innovation spirit alive.

## Eligibility

* 3+ years of Python development experience
* Strong knowledge of Django/Flask frameworks
* Experience with PostgreSQL, Redis, Celery
* Familiarity with Docker and Kubernetes
* Understanding of API design principles
