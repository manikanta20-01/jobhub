## Job Description

CloudOps Technologies helps companies implement DevOps practices. We're hiring DevOps Engineers to build and maintain CI/CD pipelines and cloud infrastructure.

You'll automate deployments, manage containerized applications, implement monitoring solutions, and ensure system reliability. Tools include Jenkins, GitLab CI, Docker, Kubernetes, Terraform, and cloud platforms.

This role combines development and operations with focus on automation and reliability.

## Eligibility

* 3+ years of DevOps or related experience
* Strong Linux administration skills
* Experience with CI/CD tools (Jenkins, GitLab CI)
* Container orchestration (Kubernetes, Docker)
* Cloud platforms (AWS/Azure/GCP)
* IaC tools (Terraform, Ansible)

## Salary Details

* Annual CTC: ₹12,00,000 - ₹24,00,000
* Cloud certification bonuses
* On-call compensation
* Conference attendance
* Stock options for senior roles

## Selection Process
