## Job Description

NetSecure Systems provides network infrastructure solutions to enterprises. We're hiring Network Engineers to design, implement, and maintain secure network environments.

You'll configure routers, switches, firewalls, and VPN solutions. Responsibilities include network monitoring, troubleshooting, security hardening, and documentation. Experience with Cisco, Juniper, and Fortinet equipment is valued.

Projects include data center migrations, SD-WAN deployments, and network security implementations. You'll work with cutting-edge networking technologies.

## Eligibility

* CCNA certification required (CCNP preferred)
* 3+ years of network engineering experience
* Experience with routing protocols (OSPF, BGP)
* Knowledge of network security practices
* Understanding of cloud networking concepts

## Salary Details

* Salary range: ₹6,00,000 - ₹12,00,000/year
* Certification bonuses
* On-call allowances
* Professional development budget
