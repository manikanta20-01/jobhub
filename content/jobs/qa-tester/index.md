## Job Description

QualityFirst Software provides testing services to product companies. We're hiring QA Testers to ensure software quality through manual and automated testing.

You'll write test cases, execute test plans, report bugs, and work with developers to resolve issues. Automation testers will create scripts using Selenium, write API tests, and maintain test frameworks.

We provide training on testing tools and methodologies. Certifications like ISTQB are valued and supported.

## Eligibility

* Any graduate with analytical mindset
* Understanding of SDLC and testing concepts
* Basic SQL knowledge
* For automation: Java/Python programming skills
* ISTQB certification is a plus
* Freshers welcome

## Salary Details

* CTC: ₹4,00,000 - ₹8,00,000 per year
* Automation testers earn higher
* Certification bonuses
* Bug bounty rewards
* Flexible working hours

## Selection Process

1. **Aptitude Test**: Logical reasoning
2. **Testing Basics**: Software testing concepts
3. **Practical Test**: Find bugs in sample application
4. **Programming Test**: For automation roles
5. **Interview**: Discussion and offer

## How to Apply

Apply with resume mentioning any testing courses, projects, or certifications. Freshers should highlight analytical abilities.
